# 智能环境监测 APP

## 功能特性

### ✅ 已实现功能

#### 1. 连接管理
- **蓝牙连接**: 扫描并连接蓝牙设备
- **WiFi连接**: 扫描并连接局域网WiFi设备
- **自动重连**: 支持连接断开后自动重连
- **设备管理**: 保存常用设备，快速连接

#### 2. 实时监控
- **温度监测**: 实时显示温度数据（-40°C ~ 80°C）
- **湿度监测**: 实时显示湿度数据（0% ~ 100%）
- **空气质量**: 实时显示AQI指数（0 ~ 500）
- **光照强度**: 实时显示光照强度（0 ~ 100000 Lux）
- **数据过期提醒**: 超过10秒未更新显示警告

#### 3. 设备控制
- **显示模块**: 远程控制显示模块开关
- **声光报警**: 远程控制报警模块开关
- **驱动模块**: 远程控制驱动模块开关

#### 4. 历史数据
- **数据存储**: 自动保存所有传感器数据到本地数据库
- **数据可视化**: 折线图展示历史数据趋势
- **统计分析**: 显示最大值、最小值、平均值
- **时间筛选**: 支持1小时、6小时、24小时、7天、30天
- **数据导出**: 导出CSV格式数据文件

#### 5. 报警功能
- **阈值设置**: 为每个传感器设置上下限阈值
- **实时报警**: 数据超标时发送系统通知
- **报警间隔**: 避免频繁通知（5分钟间隔）
- **报警开关**: 可启用/禁用报警功能

#### 6. 应用设置
- **刷新间隔**: 设置数据刷新频率（1/2/5/10秒）
- **温度单位**: 切换摄氏度/华氏度
- **主题切换**: 浅色/深色/跟随系统
- **数据保留**: 设置历史数据保留天数（7/15/30/60天）

## 技术架构

### UI架构
- **底部导航**: 三个主要页面（监控/数据/我的）
- **Fragment**: 使用Fragment实现页面切换
- **Material Design 3**: 现代化UI设计

### 数据层
- **Room数据库**: 本地数据持久化
- **LiveData**: 响应式数据更新
- **ViewModel**: MVVM架构

### 连接层
- **蓝牙SPP**: 经典蓝牙串口通信
- **WiFi Socket**: TCP/IP网络通信
- **协议解析**: 自定义二进制通信协议

### 权限要求
- 蓝牙权限（BLUETOOTH_CONNECT, BLUETOOTH_SCAN）
- 位置权限（ACCESS_FINE_LOCATION）
- WiFi权限（INTERNET, ACCESS_WIFI_STATE）
- 通知权限（POST_NOTIFICATIONS）

## 使用说明

### 首次使用
1. 授予必要的权限
2. 点击"连接设备"选择蓝牙或WiFi
3. 选择要连接的设备
4. 连接成功后即可查看实时数据

### 设置报警
1. 进入"我的"页面
2. 点击"报警阈值"
3. 设置各传感器的上下限
4. 保存设置

### 查看历史
1. 进入"数据"页面
2. 选择时间范围和传感器类型
3. 查看图表和统计信息
4. 可导出CSV数据

## 通信协议

### 数据帧格式
```
[帧头(2字节)] [长度(1字节)] [命令(1字节)] [数据(N字节)] [校验(1字节)]
```

### 命令类型
- 0x01: 传感器数据
- 0x02: 控制命令
- 0x03: 心跳包
- 0x04: 确认应答

### 传感器数据格式
- 温度: 4字节 float
- 湿度: 4字节 float
- 空气质量: 4字节 int
- 光照强度: 4字节 int
